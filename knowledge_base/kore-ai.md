# Kore.ai — Who We Are (positioning reference)

> Source: Kore.ai analyst-validation and competitive decks (June 2026). Capability
> claims are **Kore.ai's own positioning** unless attributed to a named analyst.

## One-liner
Kore.ai is an **enterprise-grade, AI-native agentic platform** for customer
experience and contact-center automation — governed, full-scope, omnichannel
(voice **and** digital), provider-agnostic on models.

## The platform
A single stack rather than point tools:
- **Contact Center AI, Agent AI, Quality AI, Outbound** running as one stack across
  voice and digital (true omnichannel — not voice-first only).
- **Voice-native**: in-house voice gateway, SSML, barge-in, sub-500ms scripted turns.
- **Integrations**: 80+ automation connectors, 250+ CRM/ERP integrations; prebuilt
  healthcare, banking and retail apps.
- **Knowledge**: permission-scoped retrieval with citations and stale-knowledge tracking.
- **A2A federation** as a first-class primitive.

## ABL — Agent Behavior Language (the core differentiator)
Kore.ai's central wedge vs. "prompt + custom code" competitors:
- **Compile-time validation** — broken handoffs, invalid tool references, unreachable
  steps and type mismatches are caught at build time, not in test or production.
- **Deterministic execution** — a zero-LLM scripted mode and compile-checked
  constraints make critical/compliance paths deterministic, not probabilistic.
- **Multi-model tiering / routing** — agents declare *tiers* not *vendors*
  (provider-agnostic, fully portable). Routing picks the right model per task by cost,
  speed and complexity: fast for the simple ~80%, reasoning for the complex ~20%.
- **Git-native release engineering** + structural pre-flight gate before deploy.
- **Platform depth**: cross-agent trace trees, 100% conversation evaluation,
  per-agent/intent/tool cost attribution.
- **Wider author base**: domain experts (not just developers) can build compile-valid
  agents; AI generates valid ABL.

## Analyst standing (Kore.ai's claim)
Recognized as a **Leader across six evaluations** — Gartner Magic Quadrant (Aug 2025),
Forrester (x2 Waves), Everest (x2 PEAK matrices), and IDC MarketScape.

## Standard competitive plays
- Against **voice-only / voice-first** rivals (PolyAI, Parloa): win on true omnichannel,
  enterprise reach, per-task model routing, and analyst validation.
- Against **conversation-data / agent-build** rivals (Cresta): concede their
  conversation-data moat gracefully, reframe it as a narrow design input, and win on
  structure, determinism, governance and platform breadth.

## Key terms / protocols the agent should understand
- **MCP (Model Context Protocol)** — how an agent accesses tools/data.
- **A2A (Agent-to-Agent)** — how agents talk to each other.
- **OKF (Open Knowledge Format)** — Google's format for storing/sharing knowledge.
- **ABL** — Kore.ai's Agent Behavior Language (above).
- Relevant analyst firms: **Gartner** (Magic Quadrant), **Forrester** (Wave),
  **Everest Group** (PEAK Matrix), **IDC** (MarketScape), **Opus Research**.
